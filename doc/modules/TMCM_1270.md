# TMCM-1270

The TMCM-1270 is a smart single-axis stepper motor driver module. The module is controlled via a CAN bus interface. Voltage supply: 6 - 24V.

## Axes

This is an enumeration of all accessible axes and its features.

### `MOTOR_0`

Index: 0

#### Features

- LinearRamp
- StallGuard2
- Current
- MotorControl
